父级爸爸其实非常的厉害。有以下属性。box-orient,box-direction,box-align,box-pack,box-lines

box-orient用来确定子元素的方向
  • inline-axis是默认值。horizontal和它的表现几乎一致，横排
  • vertical和block-axis几乎一致。竖排
box-direction,子元素的排列顺序
normal,reverse,inherit
box-align和box-pack都是决定盒子内部剩余空间怎么使用。在行为上表现为对齐。box-align是竖直对齐方式。box-pack是横向对齐方式
  • start
  • center
  • end
  • baseline 字对齐
  • stretch 也就是父标签多高，孩子元素就高
  • box-pack start
  • box-pack end
  • box-pack center
  • box-pack justify
box-lines是否换行显示。
  • single 单行显示
  • multiple 允许多行显示
